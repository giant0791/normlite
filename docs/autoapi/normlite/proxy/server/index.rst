normlite.proxy.server
=====================

.. py:module:: normlite.proxy.server

.. autoapi-nested-parse::

   Flask-based proxy server.

   This module provides a simple Flask-based RESTful proxy server to enact transactional semantics
   on Notion resources.

   .. warning:: This is part of the exploration. Prototype code, not ready for production.

   .. versionadded:: 0.6.0
