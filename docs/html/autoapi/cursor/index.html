class cursor._FrozenAttributeMixin(*, frozen_attributes: set[str] = None)

A mixin that prevents setting or deleting attributes listed in self._frozen_attributes.

Uses object-level access to avoid recursion during attribute handling. It is used by Row to implement read-only attribute for the row columns to access the corresponding values
