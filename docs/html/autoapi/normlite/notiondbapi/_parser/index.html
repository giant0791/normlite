normlite.notiondbapi._parser

Provide the parser for Notion API objects.

This module implements a simple Notion API objects parser for constructing an AST, which is used by the visitor for cross-compilation of Notion JSON objects into tuples of elements.

Important

This module is private to the package notiondbapi and it does not expose its features outside.

normlite.notiondbapi._parser.parse_number(number: dict | int | float) → Tuple[str, int | float | None][source]

Parse a number object.

This method parses the following number objects:

>>> # empty number
>>> number = None
>>> ptype, value = parse_number(number)
>>> ptype, value
('number', None)

>>> # number with format spec
>>> number = {"format": "dollar"}
>>> ptype, value = parse_number(number)
>>> ptype, value
('number.dollar', None)

>>> # number with numeric value (int or float)
>>> number = 2
>>> ptype, value = parse_number(number)
>>> ptype, value
('number', 2)
Parameters:

number (Union[dict, int, float]) – Either a dictionary containing a number spec or a numeric value.

Returns:

The pair (type, value) as tuple.

Return type:

Optional[Tuple[str, Union[int, float, None]]]
