normlite.proxy.routes

Transaction Proxy Server Routes.

This module provides the Flask routes that constitute the REST API for the transaction proxy server. All endpoints return JSON objects of the following form:

{
    "transaction_id", "<tx_id>",    // <tx_id> = uuid4 string, returned by POST /transactions only
    "state": "<tx_state>",          // <tx_state> = transaction state, returned by
                                    // POST /transactions/<id>/insert
                                    // POST /transactions/<id>/commit
                                    // POST /transactions/<id>/rollback
    "data": "<result_sets>",        // <result_sets> = a list containing all results returned by each operation committed/rolled back
    "error": "<error>"              // <error> = error message string, returned in case of error only
}
Transaction Proxy Server REST API

Method

Endpoint

Description

POST

/transactions

Begin a new transaction (see normlite.proxy.routes.transactions.begin_transaction() for more details).

POST

/transactions/<id>/insert

Add an inser operation to an existing transaction (see normlite.proxy.routes.insert.insert() for more details).

POST

/transactions/<id>/commit

Commit an existing transaction (see normlite.proxy.routes.transactions.commit_transaction() for more details).

POST

/transactions/<id>/rollback

Roll back an existing transaction (see normlite.proxy.routes.transactions.rollback_transaction() for more details).

Added in version 0.6.0.
