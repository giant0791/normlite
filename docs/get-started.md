# Get Started

> **Status:**
> This project is currently in an early stage and is **not available on PyPI yet**.

## Install `normlite`
The preferred way to install `normlite` is using [`uv`](https://docs.astral.sh/uv/).
1. Run the following command in your command line to add `normlite` to your developer dependencies:
```console
$ uv add normlite --dev
```
2. Alternatively, if you prefer to use `pip`, run the following command:
```console
$ pip install normlite
```
`normlite` is ready to be used!
