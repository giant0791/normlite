# Get Started

> **Status:**
> This project is currently in an early stage and is **not available on PyPI yet**.

## Install `normlite`
The recommended way to install `normlite` is using [`uv`](https://docs.astral.sh/uv/), but you are free to choose your favorite dependecy manager.

Run the following command in your command line to add `normlite` to your developer dependencies:

::::{tab-set}

:::{tab-item} uv
```console
$ uv add normlite --dev
```
:::

:::{tab-item} pip
```console
$ pip install normlite
```
:::

::::

`normlite` is ready to be used!
